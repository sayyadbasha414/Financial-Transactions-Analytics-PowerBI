📊 Financial Transactions Analytics Dashboard
📌 Project Overview

This Power BI project provides a comprehensive analysis of financial transactions, customer behavior, and revenue performance. The dashboard enables stakeholders to monitor transaction volumes, taxes, fees, customer segments, and business performance through interactive visualizations and dynamic metrics.

🎯 Business Objective
The primary objectives of this dashboard are:
Monitor overall financial performance.
Track transaction volume and value.
Analyze customer segments and behavior.
Evaluate transaction success rates.
Monitor tax and fee collections.
Identify high-performing transaction categories.
Support data-driven financial decision-making.
